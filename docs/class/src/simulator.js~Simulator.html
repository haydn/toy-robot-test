    Simulator

    Simulates moving a toy robot on a 5x5 table.

    Note: The 0,0 coordinate is in the south-west corner.

    Example:

    var simulator = new Simulator();

simulator.exec('PLACE 1,2,NORTH'); // ''
simulator.exec('MOVE'); // ''
simulator.exec('REPORT'); // '1,3,NORTH'
